#### Add Functionality

The app must have the following functionalities

- The `Card` component will receive the details of the card in the prop
  `cardData`
- Each card object will have the following properties

  | Key         | Data Type |
  | ----------- | --------- |
  | id          | Number    |
  | title       | String    |
  | description | String    |
  | imgUrl      | String    |
  | className   | String    |
